# Integração Bitrix24: Snippets de Cálculo de Tempo por Etapa

Este repositório armazena fragmentos de código desenvolvidos em PHP para interagir com a API do CRM Bitrix24. O foco específico destas funções é rastrear o histórico de movimentação de um lead e calcular o tempo exato de permanência em cada etapa do funil de vendas.

Este material não representa a aplicação completa, que é de propriedade privada da empresa onde o projeto foi implementado. O conteúdo aqui presente consiste exclusivamente em trechos de lógica e funções utilitárias que isolei para fins de documentação técnica e estudo.

Os exemplos cobrem a lógica de conexão via API (REST), a iteração sobre o histórico de alterações do lead e o cálculo matemático de intervalos de tempo entre as mudanças de status.
